### 2. Introduction

Open source contribution is the practice of collaborating on publicly available software projects. Anyone can view, use, modify, and improve the source code. This team practice aims to introduce members to open source culture, tools, and workflows.

---

### 3. Objectives

- Understand the concept of open source software
- Learn how to find beginner-friendly projects
- Practice using Git and GitHub
- Make at least one real contribution
- Improve teamwork and communication skills
